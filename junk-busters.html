Junk Busters Mobile Game

Game Overview

Junk Busters is a Star Wars themed Android game where players control Luke Skywalker battling against Darth Vader. The game challenges players to time their lightsaber swings perfectly to deflect space junk being thrown by Vader. With authentic Star Wars sound effects, music, and animated characters, the game delivers an immersive experience that demonstrates my ability to create engaging interactive entertainment while applying core programming concepts.

Development Technology

The game was developed using several key technologies:

  • Java - Primary programming language for Android development
  • Android SDK - Framework for creating mobile applications
  • Android Animation Framework - Native animation system for character movements and effects
  • SQLite - Local database for storing leaderboard information
  • Custom Collision Detection - Timing-based mechanics for gameplay interactions

Game Features

Junk Busters includes several engaging gameplay elements:

  • Health system for both Luke and Vader (3 health points each)
  • Progressive difficulty as player blocks more junk
  • Victory condition: Reduce Vader's health to zero by successfully blocking 15 pieces of junk
  • Defeat condition: Luke's health reaches zero from missed blocks
  • Interactive sound effects including authentic Star Wars lightsaber sounds and character dialogue
  • Customized UI with glowing buttons and character animations
  • Local leaderboard system with photo capture functionality

Technical Challenges

During development, I overcame several technical hurdles:

  • Implementing precise timing-based collision detection for junk objects
  • Creating synchronized animations with sound effects for game feedback
  • Designing an efficient game loop with proper object lifecycle management
  • Balancing difficulty progression to create an engaging player experience
  • Integrating camera functionality with the leaderboard system
  • Managing multiple media players for concurrent sound effects

Learning Outcomes

This project provided valuable experience in several areas:

  • Object-oriented programming principles applied to game development
  • Event-driven programming with Android's animation framework
  • Asset management and resource optimization for mobile platforms
  • User experience design for interactive applications
  • Testing and debugging techniques for real-time applications
  • Project planning and incremental development methodology
